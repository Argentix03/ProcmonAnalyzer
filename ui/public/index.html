About this Tool

This analyzer automatically ingests system telemetry to heuristically detect writable paths interacted with by high-privileged processes, exposing Local Privilege Escalation (LPE) and Hijacking vectors.

  1. Click the Download ETW Trace Script button on the sidebar.
  2. Open an Administrator PowerShell prompt and run: .\NativeTrace.ps1 -Start.
  3. Execute your target workflow or trigger background processes.
  4. Stop the trace: .\NativeTrace.ps1 -Stop.
  5. Drag and drop the generated NativeWritablePaths.csv file into the UI.
